# UI Package

Shared shadcn-generated UI primitives.

Most app work should not happen here. Prefer composing these primitives in
`apps/web/src/components` or `apps/web/src/features`.

## Edit Policy

- Add generated shadcn primitives here.
- Keep component names and exports close to shadcn conventions.
- Avoid domain-specific surveillance wording in this package.
- Put domain-specific components in `apps/web/src/components/interfaces`.
